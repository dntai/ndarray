Detailed Description

template<typename T>
struct ndarray::PyConverter< T >

A class providing Python conversion functions for T.

Undocumented specializations exist for bool, (unsigned) short, (unsigned) int, (unsigned) long, (unsigned long long), float, double, std::complex<float>, std::complex<double>, and std::string.
